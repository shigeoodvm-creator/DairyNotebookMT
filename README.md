# 乳検レポートアプリ

乳検速報CSV（Shift-JIS）を読み込み、牛群分析レポートをブラウザ上で自動生成するオフラインHTMLアプリです。

## 特徴

- **ファイル1つで完結** — `app.html` をブラウザで開くだけで動作します
- **完全オフライン** — インターネット接続不要、データは外部に送信されません
- **Shift-JIS対応** — NOSAI発行の乳検速報CSVをそのまま読み込み可能
- **月次データ蓄積** — 複数月のCSVを取り込んで推移グラフを表示
- **農場別データ管理** — 複数農場のデータを一括管理
- **ローカルファイル保存** — ブラウザのキャッシュをクリアしてもデータが消えない

## 動作環境

| ブラウザ | 動作 | ローカルファイル保存 |
|---------|------|-------------------|
| Google Chrome 86以降 | ✅ | ✅ |
| Microsoft Edge 86以降 | ✅ | ✅ |
| Firefox | ✅ | ⚠️ localStorage のみ（キャッシュクリアで消える） |
| Safari | ✅ | ⚠️ localStorage のみ |

**推奨ブラウザ：Google Chrome または Microsoft Edge**

## 使い方

### 初回セットアップ

1. `app.html` をChrome/Edgeで開く
2. 「データファイルを作成」ボタンをクリックし、保存場所とファイル名を指定する
   （例：`乳検データ.json`）
3. 以降、このファイルにデータが自動保存されます

### データ取り込み

1. 「CSVファイルを選択」から乳検速報CSVを選択
2. 「取り込む」ボタンをクリック
3. 「レポートを作成」ボタンでレポートを表示

### 2回目以降の起動

1. `app.html` を開く
2. 「ファイルへのアクセスを許可」ボタンをクリック（ブラウザのセキュリティ確認）
3. 前回のデータが自動的に読み込まれます

### レポート印刷・PDF保存

レポート画面の「印刷」ボタン → ブラウザの印刷ダイアログで「PDFに保存」を選択

## レポート内容

- **牛群概要** — 頭数、平均乳量、乳質（体細胞数・乳脂肪・乳蛋白）
- **乳量分布** — 乳量帯別頭数バーグラフ
- **初産牛分析** — DIM×乳量散布図（目標乳量ライン付き）
- **経産牛分析** — DIM×乳量散布図（目標乳量ライン付き）
- **異常値アラート** — ケトーシス疑い、高体細胞数、DeNovo低値など
- **月次推移** — 牛群平均乳量・体細胞数の推移グラフ
- **個体リスト** — 全頭データ一覧

## 設定

入力画面の「設定」から以下を変更できます：

| 設定項目 | デフォルト | 説明 |
|---------|-----------|------|
| 初産100日以内乳量目標 | 30 kg | 散布図の警戒ラインに使用 |
| 2産以上100日以内乳量目標 | 35 kg | 散布図の警戒ラインに使用 |

## データについて

- 取り込んだデータはブラウザ内（および指定したローカルJSONファイル）にのみ保存されます
- データが外部サーバーに送信されることは一切ありません
- JSONファイルをコピーすることで、他のPCや他のユーザーとデータを共有できます

## 対応CSVフォーマット

NOSAI北海道が発行する「検定日速報」CSVファイル（Shift-JIS、55列）に対応しています。
