# 乳検アプリ 開発ガイド

## アプリの目的

酪農家・獣医師向けの乳検（乳牛群検定）データ分析ツール。
CSVをブラウザにドロップするだけで動作する**シングルファイルHTML**アプリ。

### 核心コンセプト

**「乳検オフィシャルでは把握しにくいデータを可視化する」**

- 公式の乳検成績書では読み取りにくいパターン・傾向を、グラフ・チャートで直感的に把握できるようにする
- 全個体の羅列（それは既存の乳検成績書を見ればよい）ではなく、**問題個体・要注意傾向をいかに簡便に把握できるか**が設計の軸
- 表よりもビジュアル（散布図・円グラフ・バーチャート）を優先する

### 表示すべきもの（優先）

- 異常値・アラート個体のリストアップ（体細胞高値、乳量不足、BHB高値など）
- 群全体の傾向を俯瞰できるグラフ（DIM×乳量、DIM×LS など）
- 産次別・DIM別の比較分析
- 公式成績書では計算されない指標の可視化（P/F比、デノボFA、BHBなど）

### 表示しないもの（不要）

- 全個体の一覧テーブル（既存の乳検成績書で確認できるため不要）
- 単純なデータの転記・羅列

## 技術仕様

- **単一ファイル**: `app.html` のみ（HTML + CSS + JavaScript をすべて内包）
- **依存なし**: 外部ライブラリ・サーバー不要、ブラウザのみで動作
- **データ**: 検定日速報CSV（LIAJ形式）をドラッグ&ドロップで読み込み
- **印刷対応**: `@media print` で印刷・PDF保存に最適化済み

## 使用者

酪農専門の獣医師（ユーザー本人）が農場訪問時に使用。
